# ColumnVisibility

## 注意事项

## 输入参数
| 参数名             | 说明                           |
| :----------------- | :----------------------------- |
| hiddenFields       | 隐藏的列                       |
| disableUserControl | 禁止用户操作                   |
| options            | 列的调整设定，包括禁用用户操作 |

# 可自定义组件
| 组件名        | 说明       |
| :------------ | :--------- |
| ToolbarButton | 工具栏按钮 |

# 输入
| 输入名       | 说明   |
| :----------- | :----- |
| tableColumns | 表格列 |

# 输出
| 输出名       | 说明             |
| :----------- | :--------------- |
| tableColumns | 表格列，设定列宽 |
