# DataTypeProvider

## 注意事项

## 输入参数
| 参数名 | 说明         |
| :----- | :----------- |
| type   | 数据类型     |
| name   | 数据类型名称 |
| aligh  | 对齐方式     |

# 可自定义组件
| 组件名          | 说明             |
| :-------------- | :--------------- |
| CategoryCell    | Category的单元格 |
| CategoryContent | Category的内容   |
| Title           | 列标题           |
| Category        | 列Category       |
| Formatter       | 值的格式化       |
| Editor          | 值的编辑         |

# 输入
| 输入名       | 说明   |
| :----------- | :----- |
| columns      | 所有列 |
| tableColumns | 表格列 |
| tableRows    | 表格行 |

# 输出
| 输出名    | 说明                        |
| :-------- | :-------------------------- |
| dataTypes | 数据类型，追加/替换当前类型 |
