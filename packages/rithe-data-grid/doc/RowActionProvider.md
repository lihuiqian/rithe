# RowActionProvider

## 注意事项
- 通常为内部用，用于追加行操作

## 输入参数
| 参数名 | 说明     |
| :----- | :------- |
| key    | 操作键   |
| name   | 操作名称 |

# 可自定义组件
| 组件名 | 说明 |
| :----- | :--- |
| Action | 按钮 |

# 输入
| 输入名     | 说明     |
| :--------- | :------- |
| rowActions | 所有操作 |

# 输出
| 输出名     | 说明                        |
| :--------- | :-------------------------- |
| rowActions | 所有操作，追加/替换当前类型 |
